# Codewars

This repository contains my Python solutions for Codewars katas.

The main purpose of this repository is to keep track of solved programming exercises, practise problem-solving, and improve Python code quality over time. The solutions are grouped by Codewars difficulty level.

## Repository structure

```text
.
├── 4kyu/
├── 5kyu/
├── 6kyu/
├── 7kyu/
├── 8kyu/
└── README.md
```

## Directory description

The repository is organized by kata difficulty:

- 8kyu contains beginner-level tasks.
- 7kyu contains basic algorithmic exercises.
- 6kyu contains intermediate tasks.
- 5kyu contains more complex problem-solving exercises.
- 4kyu contains advanced kata solutions.

Each file usually contains one kata solution. Files also include a link to the original Codewars task and local **pytest tests** used to verify the solution.

## Notes

This repository is mainly intended for learning and practice. Some older solutions may be intentionally simple or reflect the implementation style used at the time of solving the kata.
